Keyboard Shortcuts

Generally speaking...

  • Command kicks off most keyboard shortcuts
  • Shift usually does the opposite (for example, if cmd + Z is undo, cmd + shift + Z is redo)
  • Option and control are usually modifiers
  • It’s totally OK to keep mashing keys until what you want happens (you > machine)
  • Keyboard shortcuts are almost always also actions available and documented in the applications main menu (for example, open the File menu)

Working with text

The following shortcuts work in most environments where you're editing text. Modify any of these commands with shift to make a selection.

Shortcut Description
opt + → Move cursor to the end of next word
opt + ← Move cursor to the beginning of previous word
cmd + → Move cursor to the end of a line or paragraph
cmd + ← Move cursor to the end of a line or paragraph
cmd + ↑ Move cursor to the beginning of file
cmd + ↓ Move cursor to the end of of file

Atom

Shortcut Description
cmd + N New file
cmd + S Save current file
cmd + D Make a selection then use this command to also highlight the next instance of what is currently selected. This shortcut is great for a quick find/replace to rename a variable or fix a typo.
cmd + return Create a new line below the current line and move your cursor there. Great to continue editing regardless of where your cursor may be.
cmd + click Creates multiple cursors at the beginning of each line clicked to type on many lines with the same text.
Any tag name (e.g, p, img) + tab HTML-only. Stubs out an empty tag for you to fill in (press tab to cycle through the parts of the element without touching your mouse)
cmd + opt + . HTML-only. Closes the last open tag

Chrome

Shortcut Description
cmd + R Refresh Page
cmd + opt + I Open the Elements tab of the Developer Tools
cmd + opt + J Open the Console tab of the Developer Tools
ctrl + tab Switch tab (forward)
ctrl + shift + tab Switch tab (backward)

Mac

Shortcut Description
cmd + ctrl + space Bring up Spotlight to search. Just start typing to find applications, contacts, documents, and more. Also you can do math, like 2 + 2
cmd + tab Switch programs (forward)
cmd + shift + tab Switch programs (backward)
cmd + ` Switch windows within a program (forward)
cmd + shift + ` Switch windows within a program (backward)
cmd + shift + 4 Take a screenshot (saved to desktop) and... free ruler!
