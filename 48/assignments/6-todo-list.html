Put some todos on the page with HTML and use an <input type="checkbox"> to make them checkable. This is a great chance to use a label and some advanced to CSS to create the desired affect. When the todo list form is submitted:

  1. Remove the element that says "Nothing to do". Can you think of how to do this? Think about what CSS property you can set that makes something disappear. Can you apply that rule via CSS?
  2. Run the code that makes a todo, using the input's value as the textContent. Nothing new here!
